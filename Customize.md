Вы можете кастомизировать тему как угодно с помощью CustomCSS
```
:root {
    --accent-1: hsl(25, 70%, 60%);
        --accent-2: hsl(25, 70%, 48%);
       --accent-3: hsl(25, 70%, 42%);
        --accent-4: hsl(25, 70%, 36%);
        --accent-5: hsl(25, 70%, 30%);
        --mention: hsla(25, 80%, 52%, 0.1);
        --mention-hover: hsla(25, 80%, 52%, 0.05);
    --text-0: white;
        --text-1: var(--text-2);
        --text-2: hsl(0, 0%, 70%);
        --text-3: hsl(0, 0%, 60%);
        --text-4: hsl(0, 0%, 50%);
        --text-5: hsl(0, 0%, 40%);
    --bg-1: hsl(25, 10%, 15%);
        --bg-2: hsl(25, 10%, 12%);
        --bg-3: hsl(25, 10%, 9%);
        --bg-4: hsl(25, 10%, 5%);
        --hover: hsla(25, 15%, 15%, 0.1);
        --active: hsla(25, 15%, 15%, 0.2);
        --message-hover: hsla(25, 10%, 8%, 0.1);
}
```
